# AUAUSAVE HOUSE Artist Website

เว็บไซต์สำหรับแฟนคลับและระบบหลังบ้านแอดมิน สร้างด้วย HTML, CSS และ JavaScript โดยไม่ต้องติดตั้ง dependency

## เริ่มใช้งาน

เปิด `index.html` ในเว็บเบราว์เซอร์ หรือรัน local server:

```powershell
python -m http.server 8080
```

จากนั้นเปิด http://localhost:8080

## ฟีเจอร์

- หน้าแรกและรายชื่อศิลปิน
- หน้าโปรไฟล์ของศิลปินแต่ละคน
- ตารางงาน รางวัล และวิดีโอ YouTube
- หลังบ้านเพิ่ม แก้ไข และลบข้อมูลได้
- ข้อมูลที่แก้ไขบันทึกใน Local Storage ของเบราว์เซอร์
- รองรับมือถือและเดสก์ท็อป

> ระบบนี้เป็น prototype ฝั่ง frontend สำหรับระบบจริงควรเชื่อมต่อฐานข้อมูล, API, ระบบล็อกอิน และจัดเก็บรูปภาพบน cloud storage
